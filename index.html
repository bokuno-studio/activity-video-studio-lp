Activity Data
Overlay Studio

.FIT アクティビティデータとアクションカメラの動画を自動同期し、心拍・ペース・GPS トラックをオーバーレイ合成する macOS アプリ。
Sync your .FIT activity data with action-camera footage and render heart rate, pace, and GPS track overlays — natively on macOS.

機能 / Features

🗺️

GPS ミニマップ / GPS Minimap

GPS トラックと現在位置を動画にリアルタイム表示。衛星画像タイルに対応。

❤️

心拍ゾーンカラー / HR Zone Colors

.FIT の心拍ゾーン設定を自動読み取り。ゾーン別にカラーコーディング。

📈

標高プロファイル / Elevation Profile

ルート全体の標高グラフに、進行インジケーターをリアルタイム描画。

🎬

大容量 4K 対応 / Large File Support

ストリーミング処理により 10 GB 超の 4K 動画も安定してエクスポート。

🔗

複数動画の自動結合 / Multi-Clip Join

アクションカメラが分割した連番 MP4 を 1 本のオーバーレイ付き動画に。

✏️

テキストオーバーレイ / Text Overlays

任意のテキストを複数追加し、フェードイン / アウト付きで合成。

Privacy First / プライバシー最優先

ActivityVideoStudio はデータを一切収集しません。すべての処理は Mac 上でローカルに完結します。

✓ No data collected ✓ No server uploads ✓ App Sandbox enabled ✓ No analytics SDKs
Read Privacy Policy

動作環境 / Requirements

  • macOS 14 (Sonoma) 以上 / macOS 14 (Sonoma) or later
  • Apple Silicon または Intel Mac / Apple Silicon or Intel Mac
  • .FIT アクティビティファイル（GPS スマートウォッチで記録） / .FIT activity file (from a GPS smartwatch)
  • アクションカメラの MP4 動画 / Action-camera MP4 footage
