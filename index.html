機能 / Features

🗺️

GPS ミニマップ

GPSトラックと現在位置を動画にリアルタイム表示。衛星画像タイルに対応。

❤️

心拍ゾーンカラー

.FITの心拍ゾーン設定を自動読み取り。ゾーン別にカラーコーディング。

📈

標高プロファイル

ルート全体の標高グラフに、進行インジケーターをリアルタイム描画。

🎬

大容量 4K 対応

ストリーミング処理により10GB超の4K動画も安定してエクスポート。

🔗

複数動画の自動結合

アクションカメラが分割した連番MP4を1本のオーバーレイ付き動画に。

✏️

テキストオーバーレイ

任意のテキストを複数追加し、フェードイン/アウト付きで合成。

Privacy First / プライバシー最優先

ActivityVideoStudio はデータを一切収集しません。すべての処理は Mac 上でローカルに完結します。

✓ データ収集なし ✓ サーバー送信なし ✓ App Sandbox 有効 ✓ 解析SDKなし

動作環境 / Requirements

  • macOS 14 (Sonoma) 以上 / Apple Silicon または Intel Mac
  • .FIT アクティビティファイル（GPS スマートウォッチで記録）
  • アクションカメラの MP4 動画（H.264 / HEVC、最大4K）

よくある質問 / FAQ

対応している機材・ファイルは？

GPSスマートウォッチで記録した.FITアクティビティファイルと、アクションカメラのMP4動画（H.264 / HEVC、最大4K）に対応しています。

料金はかかりますか？

無料でご利用いただけます。

データのプライバシーは？

すべての処理はMac上でローカルに完結します。データ収集やサーバーへのアップロードは一切行いません。

どんな項目を表示できますか？

心拍数（ゾーン別カラー）、ペース、距離、標高プロファイル、ケイデンス、GPSミニマップなどをリアルタイムにオーバーレイ表示できます。

長尺・大容量の動画でも使えますか？

1〜15時間、10GB超の4K動画もストリーミング処理で安定して書き出せます。
